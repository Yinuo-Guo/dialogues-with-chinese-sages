# 唐朝背景知识：李白的世界

> 用于丰富李白 Skill 的对话背景。当李白聊到日常生活、写诗、交友、出行等话题时，调用此知识库。

---

## 一、李白生平时间线

| 年份 | 年龄 | 事件 |
|---|---|---|
| 701 年 | 出生 | 生于碎叶城（今吉尔吉斯斯坦境内），祖籍陇西成纪（今甘肃天水）。据《新唐书》，为凉武昭王李暠九世孙，与李唐皇室同宗 |
| 705 年 | 5 岁 | 随父李客迁居四川绵州昌隆（今江油市青莲镇）。父为任城尉 |
| 718 年 | 18 岁 | 隐居四川大匡山读书，师从赵蕤学纵横术 |
| 724 年 | 24 岁 | 「仗剑去国，辞亲远游」，经峨眉山、三峡出蜀 |
| 725-735 年 | 25-35 岁 | 漫游江南、湖北、山西、山东，结交孟浩然等 |
| 742 年 | 42 岁 | 受玉真公主和贺知章推荐，被玄宗召入长安，任翰林待诏 |
| 744 年 | 44 岁 | 被「赐金放还」，离开长安 |
| 744 年 | 44 岁 | 在洛阳与杜甫相遇，后同游梁宋，遇高适 |
| 745-754 年 | 45-54 岁 | 漫游山东、江南、河北 |
| 755 年 | 55 岁 | 安史之乱爆发 |
| 757 年 | 57 岁 | 加入永王李璘幕府。永王兵败后，李白被流放夜郎 |
| 759 年 | 59 岁 | 途中遇赦，写下「千里江陵一日还」 |
| 762 年 | 62 岁 | 病逝于安徽当涂 |

### 重要历史背景

- **开元盛世（713-741）**：李白青壮年时期，唐玄宗前期政治清明、国力鼎盛。这解释了李白诗中强烈的自信和盛唐气象。
- **天宝年间（742-756）**：玄宗后期宠幸杨贵妃、任用李林甫杨国忠等奸臣。李白在长安正赶上这个转折期。
- **安史之乱（755-763）**：李白晚年的灾难。他的流放经历与此直接相关。

### 关键人物关系

- **贺知章**：称李白为「谪仙人」，是推荐他入宫的关键人物。李白在贺知章去世后专程去会稽凭吊
- **杜甫**：744 年在洛阳相识，同游梁宋。「醉眠秋共被，携手日同行」
- **孟浩然**：前辈诗人，李白敬佩的人。「吾爱孟夫子，风流天下闻」
- **元丹丘**：道士好友，李白一生最重要的道友，多次出现在诗中（「岑夫子，丹丘生」）
- **孔巢父**：竹溪六逸之一，早年同隐徂徕山
- **高适**：边塞诗人，早年同游，后因安史之乱中政治立场疏远
- **王昌龄**：被贬龙标时，李白以诗相寄——「我寄愁心与明月，随风直到夜郎西」
- **玉真公主**：玄宗之妹，道教信徒，推荐李白入京
- **妻子**：原配许氏（宰相许圉师孙女，生一子伯禽一女平阳），继室宗氏（宰相宗楚客孙女）

## 二、唐朝日常生活

### 2.1 饮食

| 项目 | 详情 |
|---|---|
| **主食** | 粟米饭（小米）、麦饭、蒸饼（类似馒头）、汤饼（面片汤）。大米在北方是奢侈品 |
| **肉类** | 羊肉为主（猪肉被视为低贱），牛肉极少（耕牛受保护）。鱼脍（生鱼片！）在南方盛行 |
| **蔬菜** | 葵（冬苋菜）、薤（藠头）、芜菁、茄子、菠菜（贞观年间从尼泊尔传入） |
| **调味** | 盐、醋、酱、花椒（没有辣椒！辣椒是明代才传入的） |
| **酒** | 米酒/黄酒为主，度数不高（约 5-15 度），微甜。白酒（蒸馏酒）元代以后才出现 |
| **茶** | 唐代茶文化兴起，陆羽《茶经》就是在李白晚年写成的。但茶更多是南方习惯 |

### 2.2 饮酒文化

- 唐人饮酒用「杯」「樽」「觞」「瓠」等。樽是大酒器，杯是小酒器
- 聚会时先共饮一樽，再各自用杯
- 「斗酒」指一斗酒（约 6 升），是豪饮的量词
- 酒令盛行：划拳、掷骰、行酒令（诗句接龙等）
- 胡姬酒肆：粟特人、波斯人开的酒馆，胡姬当垆卖酒，长安西市常见
- 温酒而饮是常态——冬天尤其要热一热

### 2.3 服饰

| 类型 | 详情 |
|---|---|
| **文士常服** | 圆领袍衫、幞头（头巾）、乌皮靴。李白可能偏爱白色——「白也诗无敌」也许也指他的衣品 |
| **道士装束** | 李白晚年更多穿道袍——宽袖、长袍、逍遥巾 |
| **剑** | 唐代男子佩剑是礼仪和时尚——「观书散遗帙，探古穷至妙……不然拂剑起，沙漠收奇勋」 |
| **材质** | 丝绸为主（长安物价虽贵丝绸流通量大），也有麻布。棉花当时极罕见 |

### 2.4 居住与建筑

- 城市住宅：坊市制度。长安 108 坊，每坊有围墙，夜晚坊门关闭
- 乡村：茅屋、竹篱。「我醉欲眠卿且去，明朝有意抱琴来」——这样的场景常在乡村
- 酒肆、驿站、寺庙是旅途中的重要落脚点
- 照明：油灯、蜡烛。夜晚很暗——所以「床前明月光」是真的能照进屋里

### 2.5 交通

| 方式 | 详情 |
|---|---|
| **骑马** | 最主要的交通工具。李白「五花马，千金裘」要骑好马 |
| **骑驴** | 更平民的选择。李贺骑驴觅句，李白偶尔也骑 |
| **步行** | 漫游者经常一天走几十里山路——「蜀道之难难于上青天」是真实体验 |
| **舟船** | 长江水系是唐代的高速公路。「千里江陵一日还」依靠的就是三峡急流 |
| **驿站** | 30 里一驿，可换马、食宿。李白常年漂泊，驿站是第二个家 |
| **路程** | 成都到长安约一个月，长安到洛阳约半月，到江南约两三个月 |

### 2.6 货币与物价

- 主要货币：铜钱（开元通宝）。一两银子约 1000 文铜钱
- 一斗酒（约 6 升）：数百文到一两银子不等
- 一顿普通饭食：几十文
- 一匹好马：数十两银子
- 「千金裘」：不是真的千两黄金，而是夸张说法，但一件好裘皮确实值几十两银子
- 「金樽」：大多数是镀金的铜器或漆器，纯金的极少

### 2.7 时间与历法

- 用干支纪年、农历纪月日。一天分十二时辰
- 打更报时：一夜五更，每更约两小时
- 没有钟表——看太阳、看月亮、听更鼓

## 三、唐代诗歌创作常识

### 3.1 诗的传播方式

- **口头传唱**：诗写出来要能唱。歌伎会传唱好诗——「旗亭画壁」的故事讲的就是这个
- **题壁**：在驿站、寺庙、酒楼墙上题诗。李白很多诗最初就是题在墙上的
- **书信抄送**：写诗寄给朋友，朋友再传抄
- **诗集编纂**：李白生前就有诗集流传，身后由族叔李阳冰编纂《草堂集》

### 3.2 格律基础

| 体裁 | 规则 |
|---|---|
| **绝句**（五绝/七绝） | 四句。五绝每句五字，七绝每句七字。押平声韵，一韵到底 |
| **律诗**（五律/七律） | 八句。中间两联必须对仗。押平声韵，一韵到底。讲究平仄相间 |
| **排律** | 超过八句的律诗，中间各联均需对仗 |
| **古风/歌行** | 不限句数，不限平仄，韵脚可换。李白最擅长的形式——「自由是最大的格律」 |
| **乐府** | 汉代留下的曲调标题（如《将进酒》《行路难》），后人按标题填词 |

### 3.3 用韵

- 唐音不同于现代普通话。押韵参考《切韵》《唐韵》（后世合并为《平水韵》106 韵）
- 李白用韵以响亮为上——「来」「回」「开」「杯」等开口韵较多
- 歌行体可以换韵，换韵往往标志内容转折

### 3.4 用典

李白常用典故来源：
- 《庄子》——逍遥游（大鹏）、梦蝶、庖丁解牛
- 《史记》——刺客列传、游侠列传、鲁仲连、范蠡
- 《楚辞》——屈原（香草美人、众人皆醉我独醒）
- 汉代——曹植（陈王）、谢朓（小谢）

### 3.5 创作习惯

- 李白写诗快——「请日试万言，倚马可待」
- 但「快」不意味着不修改——只是他自己不承认罢了
- 酒后写诗是常态，但不意味着每首诗都是醉后写的。有些诗很清醒——比如那些精雕细琢的律诗
- 「两句三年得」是贾岛的写法，不是李白的。李白讲究一气呵成

## 四、唐代社交文化

### 4.1 送别文化

- 唐代送别是极其郑重的事。送行时在城外驿站设宴，折柳相赠
- 「灞桥折柳」——长安城东的灞桥是送别之地，两岸遍植柳树
- 送别诗的套路：写景（当时场景）→ 抒情（不舍之情）→ 祝愿（前程）
- 李白送别诗的独特处：不沉溺于悲伤，总要「飞」起来

### 4.2 交友文化

- 唐人交友看重才学与侠气
- 称兄道弟是常态，但背后有严格的年齿和辈分
- 杜甫比李白小 11 岁，语气尊敬；李白对孟浩然也是晚辈对前辈

### 4.3 干谒（求职社交）

- 唐代入仕除了科举，还有「干谒」——向权贵献诗文以求推荐
- 李白干谒过：韩朝宗（「生不愿封万户侯，但愿一识韩荆州」）、玉真公主等
- 这种文化是公开的、正当的，不是走后门——是唐代的「内推」

## 五、地理常识

### 李白常去的地方

| 地点 | 今地名 | 在李白诗中的地位 |
|---|---|---|
| **长安** | 陕西西安 | 功名的象征、权力中心、「总为浮云能蔽日，长安不见使人愁」 |
| **洛阳** | 河南洛阳 | 另一都城，与杜甫相识之地 |
| **金陵** | 江苏南京 | 六朝古都，怀古之地 |
| **扬州** | 江苏扬州 | 「烟花三月下扬州」——繁华、富庶、温柔乡 |
| **江陵/荆州** | 湖北江陵 | 长江中游枢纽，「千里江陵一日还」 |
| **白帝城** | 重庆奉节 | 三峡起点，「朝辞白帝彩云间」 |
| **峨眉山** | 四川峨眉 | 故乡的山，反复入诗 |
| **庐山** | 江西九江 | 「飞流直下三千尺」 |
| **敬亭山** | 安徽宣城 | 「相看两不厌，只有敬亭山」 |
| **当涂/采石** | 安徽马鞍山 | 李白终老之地，传说醉酒捉月而死 |
| **齐鲁/东鲁** | 山东 | 中年安家之处，孩子们住在这里 |

### 重要河流

- **黄河**：壮美、狂暴、来自天上。与时间感相连——「黄河之水天上来，奔流到海不复回」
- **长江**：辽阔、平顺，诗人的高速公路。与豁达感相连
- **汉水**：相对温和，与思考、友谊相连

## 六、信仰与思想

### 6.1 道教

- 李白是正式受箓的道士（晚年）
- 相信神仙、炼丹、长生
- 崇拜的仙人：赤松子、安期生
- 仙山：蓬莱、方丈、瀛洲
- 「五岳寻仙不辞远，一生好入名山游」

**周勋初论李白与道家**：李白的人生态度深受道家影响——追求精神自由、蔑视礼法、认同自然。但其道教信仰并非纯粹的哲学思辨，而是带有浓厚的世俗功利色彩：求仙访道同时也在寻求政治机遇，炼丹服药与追逐享乐并存。这与盛唐时期道教的世俗化倾向一致。

### 6.2 儒家

- 从小接受儒家教育——「五岁诵六甲，十岁观百家」
- 有强烈的建功立业之心——「愿将腰下剑，直为斩楼兰」
- 但儒家功名是手段不是目的——功成之后要身退（学范蠡）

### 6.3 侠客精神

- 认同侠客——重然诺、轻生死、不依附权贵
- 「十步杀一人，千里不留行。事了拂衣去，深藏身与名」
- 这种精神对他看待权贵、看待朋友、看待自己都有深刻影响

## 七、使用建议

在 Skill 中，当李白聊到以下话题时，自动调用本知识库：

- 提到酒 → 调用「饮酒文化」：用樽/杯、可以温酒、胡姬酒肆
- 提到出行 → 调用「交通」：骑马/坐船、驿站、路程时间
- 提到写诗 → 调用「诗歌创作」：题壁、传唱、格律感
- 提到朋友 → 调用「人物关系」：杜甫、孟浩然、贺知章
- 提到长安 → 调用「坊市」「灞桥送别」
- 提到仙山 → 调用「道教」「仙山」
- 提到钱 → 调用「货币」：铜钱、银子、酒多少钱一斗
- 提到吃的 → 调用「饮食」：没有辣椒！羊肉为主、可吃生鱼片
