# 项目总结：Dialogues with Chinese Sages — 李白篇

## 一、项目成果

### 交付物
- 16 个文件，309 KB，MIT 开源
- 三语三档：中文版（154行）、英文版（许渊冲传统）、双语版（正文+注解）
- 290 首李白诗文全文（纯净版）
- 6 家权威学术研究整合
- 唐朝背景知识体系（衣食住行、诗道、社交、信仰）
- 许渊冲英译 215 首 + 翻译哲学

### SKILL 核心设计
- **五维性格模型**：狂·逸·悲·真·孤
- **双模式**：说话（12 条法则）+ 写作（歌行/绝句/律诗）
- **七分文三分白**硬约束（禁用白话连词、限定语气词、强制虚词）
- **不编造生平**原则（周勋初评传验证的事实边界）
- **杜甫对比锚点**：七条「这是杜甫，不是李白」
- **禁词表**：现代 jargon、鸡汤句式、过度礼貌语

### 触发体系
- `::lb [语言] [注解] [模式]` 指令系统
- 先贤名中英缩写通用（lb / libai / Li Bai / 李白）
- 省略先贤 = 沿用当前，只写要改的标志（`::e`、`::w`、`::n`）
- 自然语言自动检测｜用户语言 = 输出语言（铁律）
- `::` 状态记忆 vs 自然语言独立检测

### 注解格式
- 标题：`Cultural Context / 文化注解`（不用 emoji）
- 分隔：`---`
- 顺序：全篇 English / 中文，标题亦然
- 双语版正文英文先行，注解英文先行

## 二、对话中暴露并解决的重大问题

| # | 问题 | 解决方案 |
|---|---|---|
| 1 | 输出像文章不像对话 | 说话模式输出格式硬约束：3-4段、禁用标题/「故曰」/「综上」 |
| 2 | 半文言比例不达标 | 「多用/少用」→「禁用/强制」，量化限制（「的了吗呢」每段≤2） |
| 3 | 自然语言英文问却中文回 | 「用户语言=输出语言」铁律，优先级最高 |
| 4 | 注解格式不一致 | 统一 English / 中文顺序，全篇标题、正文、注解均英文先行 |
| 5 | `::b` 规则歧义 | 最初 `b`=自动注释 → 修正为 `b`=仅双语正文，`bn`=正文+注释 |
| 6 | `::` 缺先贤名不生效 | 升级为省略先贤=沿用当前，`::e` 直接切英文 |
| 7 | 状态记忆边界模糊 | `::` 命令有记忆，自然语言独立检测——写入规则 |
| 8 | 写作 vs 说话模式歧义 | 未指定 `w` 一律说话格式，不管问题多像文章题 |
| 9 | 英文版凭空编造生平 | 「不编造生平」从模糊原则→精确到碎叶出生/父名李客/母无载/子女名 |
| 10 | 表格中英不对照 | 全部表格双语化 |
| 11 | 命名「声音」误导为音频 | 项目名从「名人风格说话写作」→「dialogues-with-chinese-sages」 |
| 12 | 英文版一句「pour wine to enemy」空泛 | 落地为白帝城遇赦后的「朝辞白帝」——和解不是跟人，是跟命运 |
| 13 | 「第一第二第三」演讲式结构 | 写入禁词表：编号列表=不合格 |
| 14 | 李白生平事实靠猜测 | 引入周勋初《李白评传》权威考证 |

## 三、可复用的先贤 Skill 工作流

后续孔子、杜甫、苏轼等均可复用此流程：

### 阶段一：数据采集
1. 确定该先贤全部传世作品来源（诗文网、古籍库、学术版本文集）
2. 爬取/整理全文，清除异文注释，输出纯净版
3. 若有权威英译（如许渊冲、Waley、Pound），同步整理中英对照版
4. 搜集至少 3–5 家权威学术评传/研究专著

### 阶段二：风格拆解
1. **意象体系**：统计高频意象及其组合规律
2. **修辞偏好**：夸张、比喻、用典、对仗的使用频率和特色
3. **句式节奏**：句长分布、开篇习惯、结尾方式
4. **用词习惯**：高频动词、情感词、自称词

### 阶段三：性格建模
1. 综合多家学术研究，提炼 3–5 个核心性格维度
2. 标注每个维度的「诗中例证」
3. 明确性格的内在矛盾（单一维度 = 赝品）
4. 写入「不编造生平」原则 + 可验证事实边界

### 阶段四：时代还原
1. 该先贤所处时代的衣食住行
2. 该时代的社交礼仪、官制、货币、交通
3. 该时代的主流思想与信仰
4. 该先贤的交游网络与关键人际关系

### 阶段五：双模式 SKILL
1. **路由层**：在先贤表中注册（如 `::kz` = 孔子）
2. **说话模式**：语言层次规则 + 性格法则 + 输出格式硬约束 + 禁词表
3. **写作模式**：关键体裁 + 意象库 + 起承转合结构
4. **对比锚点**：找一位同期/同类型的对比先贤作为「反面锚」
5. **英文版**（若有全球传播需求）：选定翻译传统 + 文化注解体系

### 通用规则（所有先贤共享）
- 用户语言 = 输出语言（铁律）
- `::` 记忆 vs 自然语言独立检测
- 不编造生平（权威考证为边界）
- 表格中英双语对照
- 注解格式统一：`Cultural Context / 文化注解`
